This plugin is used to connect seynd account with wordpress site. 

== Description ==
Seynd Connection is used to connect seynd account with wordpress site. It allow users to choose any site and place javascript code automatically.

[Main Website](https://pushnotifications.getinstafy.com)
Above is our main application url. We are using our application API in plugin to authorize user and return sub-domains in response.

[External Website js](https://pushnotifications.getinstafy.com/src/push.js)
[External Website css](https://pushnotifications.getinstafy.com/src/push.css)

By using above css and js link we are providing a popup service to front-end users in order to subscribe for web push notification. Our customer can customize popup layout and other things in application that can be implemented by above External website javascript.

Features of the plugin include:

    * Connect/Disconnect Seynd Account

    * Choose any web-site created in account.

    * Implement javascript code on selected pages or whole wordpress

    * Change site anytime.
    
    * Disconnect account anytime    

== Screenshots ==

1. Main screen to connect the seynd account.
2. If you do not have an account, click on the SignUp button to get a free account.
3. If you have an account, Enter you login information to connect the account.
4. After a successful login, a select the website that you want to connect with your WordPress.
5. Once connection established, you can change the website or select particular pages at any time
6. Popup according to selection at backend is showing on the frontend.

== Installation ==
From your WordPress dashboard
1. Visit ‘Plugin > Add new’.
2. Search for ‘Seynd Connection’.
3. Activate Seynd Connection from your Plugins page.

From WordPress.org
1. Search, select and download "Seynd Connection".
2. Activate the plugin through the ‘Plugins’ menu in WordPress Dashboard.


From WordPress.org
1. Search, select and download "Training".
2. Activate the plugin through the ‘Plugins’ menu in WordPress Dashboard.
